# Ministry of Interior Clubs App – UI/UX Case Study

##  Project Overview
A role-based mobile application designed to streamline club bookings, event management, and administrative operations for users, members, and administrators.

##  Timeline
Completed within **1 week** as part of a client’s Final Year Project (FYP).

---

##  Design Highlights
- Mobile-first approach
- Scalable design system
- Consistent spacing & typography
- Accessibility-focused UI

---

##  Problem Statement
The existing system lacked a centralized and efficient solution for:
- Booking management
- Role-based user access
- Notifications and updates
- Administrative control

This resulted in poor user experience and manual operational overhead.

---

##  Solution
Designed a scalable and intuitive UI system that enables:
- Seamless event and table booking system
- Role-based access control (Guest, Member, Admin)
- Real-time notifications
- Admin dashboard for full management control
